## 📁 저장소 개요  
- 이 저장소는 “2025-2학기 GameProgramming 수업 자료”를 주차별로 정리한 것이다.

## 주차별 수업 내용 요약  

| 주차 | 주요 학습 주제 |
|------|----------------|
| **1주차** | 개발 환경 세팅 및 기본 예제, 콘솔 입출력 기초 |
| **2주차** | 커서 제어, 화면 지우기; 키 입력 처리 및 메뉴 구현; 문자 기반 도형 출력 |
| **3주차** | 2D 배열 기초; 맵 출력 및 기본 충돌 개념; 캐릭터 이동 기초 |
| **4주차** | 구조체(struct) 활용; 캐릭터 정보 관리; 간단한 게임 루프 구조 설계 |
| **5주차** | 객체(Object) 개념 도입; 간단한 클래스 설계; 플레이어 및 적 캐릭터 분리 |
| **6주차** | 적 AI 기초: 랜덤 이동, 추적 로직; 충돌 감지 심화 |
| **7주차** | 맵 에디터 기초; 파일 입출력(File I/O) 활용; 맵 저장 및 불러오기 기능 구현 |
| **8주차** | 총알 / 투사체 구현; 적 공격 패턴; 간단한 전투 시스템 구현 |
| **9주차** | 게임 전체 구조 통합; 엔딩 조건 설정, 점수 시스템; 최종 미니 프로젝트 구성 |
| **10주차** | *(week10 폴더 존재 — README 요약 내용 없음)* |
| **11주차** | *(week11 폴더 존재 — README 요약 내용 없음)* |

## 활용 팁  
- 각 주차 폴더 안의 코드를 열어보면, 그 주차에서 학습한 개념이 어떤 식으로 구현됐는지 확인 가능  
- 구조체 → 클래스 → AI → I/O → 전투 → 전체 게임 구조로 이어지는 학습 흐름이 명확  
- 미니 프로젝트(9주차)가 전체 복습 구조로 가장 중요함
